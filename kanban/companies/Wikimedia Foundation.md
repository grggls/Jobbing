## Job Description
Senior Site Reliability Engineer, Service Operations
Design and run infrastructure and services forming the base of Wikimedia Foundation's projects, including Kubernetes clusters, application servers, datastores, and developer-facing services. Participate in 24/7 incident response and on-call rotation. Frequent collaboration with other SRE team members, Security, Release, and Software Engineers.
Technical Stack:
- MediaWiki instances running on bare metal, surrounding microservices on Kubernetes
- In-house designed deployment pipeline
- Automation using primarily Python, Puppet, and Spicerack
Required Skills:
- Proficiency with shell and a programming language in SRE/Operations context (Python, Go, Ruby, etc.)
- Comfort with open source configuration management and orchestration tools (Puppet, Ansible, Terraform)
- Experience with Linux systems administration
- Experience with Hadoop or large-scale data infrastructure
- Experience with containers and container orchestration (Kubernetes)
Compensation: $109,047 to $169,455 (US), adjusted for non-US locations
Remote position. Wikimedia Foundation is a non-profit organization.
